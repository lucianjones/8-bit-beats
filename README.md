## Project Summary
	8 Bit Beats is a website where you can browse, stream, upload, and download all of your favorite retro tracks.
## How to Install

 1. clone repo
 2. npm install
 3. set up dotenv
 4. create, migrate and seed db
 5. npm start
 6. localhost:3000
